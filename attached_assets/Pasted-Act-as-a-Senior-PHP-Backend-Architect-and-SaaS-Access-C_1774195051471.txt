Act as a Senior PHP Backend Architect and SaaS Access Control Engineer.

Build a **complete Admin Role Management System** for ClearOrbit using PHP + MySQL (NO Node.js).

---

## 🎯 OBJECTIVE

* Super Admin can create Managers
* Managers get LIMITED access
* UI + Backend both must enforce role-based access

---

## 👑 HOW MANAGER IS CREATED (IMPORTANT)

Create a page:

create_admin.php

Form fields:

* Name
* Email
* Password
* Role (dropdown)

Dropdown options:

* Super Admin
* Manager

When form submits:

* Save in "admins" table
* Password hashed
* Role saved (super_admin / manager)

---

## 🗄️ DATABASE STRUCTURE

admins table:

* id
* name
* email
* password
* role (super_admin / manager)
* created_at

---

## 🔐 LOGIN SYSTEM

On admin login:

* Store in session:
  $_SESSION['admin_role']
  $_SESSION['admin_id']

---

## 🧠 ROLE-BASED REDIRECTION

After login:

IF role == super_admin:
→ redirect to:
/admin/dashboard.php

IF role == manager:
→ redirect to:
/manager/dashboard.php

---

## 📁 SEPARATE DASHBOARDS (IMPORTANT)

1. Super Admin Panel:
   /admin/

   Pages:

   * dashboard.php
   * users.php
   * subscriptions.php
   * payments.php
   * cookies.php
   * pricing.php

2. Manager Panel:
   /manager/

   Pages:

   * dashboard.php
   * users.php ONLY

## 🚫 ACCESS CONTROL (CRITICAL)

On EVERY page:

Check role:

IF manager tries to access admin pages:
→ redirect to:
/access_denied.php

/admin/payments.php
