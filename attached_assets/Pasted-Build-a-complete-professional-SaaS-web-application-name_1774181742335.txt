     * Shared number

9. Activity Logs:

   * Track all actions:

## 🌐 EXTENSION LOGIC (IMPORTANT)

When user clicks “Access Now”:

1. Clear old cookies for that domain
2. Inject only that platform cookies
3. Open correct website

Example:

if platform == "coursera":
clear cookies for coursera.org
inject coursera cookies
open coursera.org

⚠️ Ensure:

* No cross-platform cookie mixing
* No wrong site opening (fix Udemy issue)
