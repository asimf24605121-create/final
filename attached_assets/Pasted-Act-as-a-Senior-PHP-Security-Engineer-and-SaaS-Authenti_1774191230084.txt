## 🔐 LOGIN DEVICE CONTROL

1. Each user has a device limit:

* Default: 1 device
* Max allowed: 2 devices

2. On login:

* Generate unique device_id
  (combine: IP + user-agent + random hash)

* Store in database

3. Check:

IF active devices >= allowed:
→ BLOCK login
→ Show message:
"Access denied. This account is already active on another device."

---

## 🔄 LOGOUT SYSTEM (IMPORTANT)

When user logs out:

* Remove device_id from database
* Free the slot

Result:
User can login from another browser/device

---

## 📱 DEVICE DETECTION SYSTEM

Detect device type using user-agent:

* Mobile
* Desktop (PC)
* Tablet

Save in DB:

* device_type
* browser
* OS

## 🧠 SESSION TRACKING TABLE

Create table: user_sessions

Fields:

* browser
